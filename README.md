# Utools 文本生成器插件

一个简单实用的Utools插件，用于将复制的文本快速生成txt文件并保存到桌面。

## 功能特点

- 🚀 快速将剪贴板文本保存为txt文件
- 📁 自动保存到桌面
- 🕒 自动生成带时间戳的文件名
- 🔍 支持文本预览
- ⚡ 避免文件名冲突，自动重命名

## 安装方法

1. 将整个插件文件夹复制到Utools插件目录
2. 在Utools中进入插件管理
3. 找到"文本生成器"插件并启用

## 使用方法

1. 复制任意文本内容到剪贴板
2. 打开Utools (Alt + Space)
3. 输入"生成文件"或直接看到自动显示的命令
4. 点击命令即可将文本保存为txt文件到桌面

## 文件命名规则

生成的文件名格式：`文本_YYYYMMDD_HHMMSS.txt`

例如：`文本_20240109_143025.txt`

如果文件名已存在，会自动添加数字后缀避免冲突。
